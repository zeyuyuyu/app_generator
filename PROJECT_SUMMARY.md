## 🚀 部署和使用

### 快速开始
```bash
# 克隆项目
git clone https://github.com/zeyuyuyu/app_generator.git
cd app_generator

# 快速启动
./scripts/quick-start.sh

# Web界面
npm run web
# 访问 http://localhost:3001
```

### 生产部署
```bash
# Docker部署
docker-compose -f docker-compose.prod.yml up -d

# 本地部署
npm install
npm run build
npm run web
```
